# 🚀 Refact Studio

A modern agency-style web application focused on clean UI, responsive design, smooth animations, and scalable frontend architecture.

## 🌐 Overview

Refact Studio is a modern web experience designed to showcase a professional digital agency presence.
The project focuses on performance, responsiveness, reusable components, and polished user experience across all devices.

Built with a modern frontend stack, the application demonstrates production-ready UI implementation and clean component organization.

---

# ✨ Features

* Modern responsive UI
* Mobile-first design
* Smooth animations and transitions
* Reusable component architecture
* Fast and optimized performance
* Clean layout structure
* Interactive sections
* Professional agency-style design

# 🚀 Getting Started

## Installation

```bash
git clone <repository-url>
```

```bash
cd refact-studio
```

```bash
npm install
```

## Run Development Server

```bash
npm run dev
```
